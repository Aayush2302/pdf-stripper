100% client-side · Python in browser

01
The Problem
Beamer/LaTeX decks export each slide multiple times — one page per bullet reveal. A 45-slide deck can become 158+ bloated pages.
02
Detection
Groups pages by the slide number in the footer (e.g. "6 / 45"). All pages sharing a slide number are incremental builds of that one slide.
03
The Result
Keeps only the LAST page per group — the fully-revealed version with all bullet points. Every earlier partial build is cleanly stripped out.
